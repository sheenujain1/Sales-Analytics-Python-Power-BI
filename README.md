# Sales-Analytics-Python-Power-BI
This project is an interactive Sales Analytics Dashboard developed using Power BI and Python to analyze sales performance, profitability, regional trends, and product-level insights.

The dashboard helps businesses monitor key metrics and make data-driven decisions through interactive visualizations and KPI tracking.

Tools and Technology used are:
Power BI
Python
Time intelligence function
KPI
DAX
Data cleaning and Visualization

Dataset used:
Superstore sales dataset from Kaggle
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final
